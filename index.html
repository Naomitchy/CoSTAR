Message

あなたは今、自分の人生の「主役」を生きていますか？

周りの期待に応えてきた。社会のルールも守ってきた。十分な給与も得られた。
でも、いつの間にか、自分は誰と仲が良いのか分からなくなった。
気軽に「今日、ご飯食べに行かない？」なんて、言えなくなっていた。

心の奥底で知っている。本当の自分は不真面目で自分勝手。
相手の為と見せかけて、保身や損得勘定で動くズルい自分。

パートナーとの関係さえも、役割を演じることになってしまった。
誰にも本音を言えない。建前の中で生きる日々は、もう苦しい。

「残りの人生、このまま偽りの中で生き続けるの？」
「このまま死んでいいの？」
「一体自分は何者なのか？」

私はこの問いの答えを探す旅の中で、コーチングと出会いました。
そして気づいたのです。私が「価値のない自分」だと否定し、隠してきた部分こそが、
実は人生を動かす最もパワフルなエネルギーであることを。

あなたの「完璧な仮面」の裏にある、不真面目で、自分勝手な一面も、あなた自身の全てです。その全てが、あなたが本当に望む未来を創造するためのエネルギーになります。

もしあなたが、「本当に大切にしたい自分」を生きることに飢えているなら。
恐れも、弱さも、すべてを燃料に変えて、一緒に遠くまで行きましょう。

"If you want to go fast, go alone.
If you want to go far, go together."
（意訳：道中楽しもう。欲しいものより先に、大切なものが来るから。）

Value of Coaching

気持ちがいいだけのコーチングでは、
人生は変わらない。

コーチングの時間は、日常の喧騒から切り離された、あなたにとって心安らぐ非日常の場です。
自分の枠を外し、新しい自分との出会いは、本質的な変化への最初の一歩です。

一方で、社会的成果と現実的な変化を求めるあなたは、知っています。気持ちのよいセッションを終えても、日常の喧騒や社会の論理に戻れば、すぐに元通りになるということを。

コーチングの時間は冒険（人生）における『休憩と作戦会議』だと考えています。
ここは、あなたの隠された本質的なエネルギーを再充電し、日常という「現実の舞台」へ持ち帰るための戦略（アクション）を決める場です。

「180度の変化」より、
確かな「2度の変化」を。

一度のセッションで劇的に180度変化することは、ほとんどないと考えています。

作戦会議で決めた、今までのあなたでは出来なかった180度の行動を現実で試してみる。 実践できた自分は誇らしく、確かに変わった気がするでしょう。

しかし、時間がたてばたつほど、今まで慣れ親しんだ過去の自分が、必ず「揺れ戻し」をします。

コーチングの価値は、人生の角度を「2度」変えることにあります。
揺れ戻しを乗り越えるための伴走と軌道修正。
この地道な「2度」の積み重ねが、振り返った時に、あなたの人生を「深く、大きく」変化させているのです。

2. 個人情報の収集方法

当方は、ユーザーが本サービスへの問い合わせや申し込みをする際に、氏名、生年月日、住所、電話番号、メールアドレス、職務経歴、現在の悩みなどの個人情報をお尋ねすることがあります。

3. 個人情報の利用目的

当方が個人情報を収集・利用する目的は、以下のとおりです。

  • 本サービスの提供・運営のため

  • メンテナンス、重要なお知らせなど必要に応じたご連絡のため
  • 上記の利用目的に付随する目的

4. 個人情報の第三者提供

当方は、次に掲げる場合を除いて、あらかじめユーザーの同意を得ることなく、第三者に個人情報を提供することはありません。ただし、個人情報保護法その他の法令で認められる場合を除きます。

  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
